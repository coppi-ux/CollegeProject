## Projects

### Library Management System
A console-based library system built in C implementing core data structure concepts.

**Features:**
- User registration and login
- Book/instrument catalog management
- Rental and return tracking
- File-based persistence (users, rentals, inventory)

**Files:**
| File | Description |
|------|-------------|
| `FinalAOL_DS.c` | Final submission — full system |
| `dataStructAOL.c` | Data structures implementation |
| `dataStructAOLtime.c` | Version with time-based tracking |
| `librarySystem` | Compiled binary |
| `librarySystem_new` | Updated compiled binary |
| `instruments.txt` | Instrument catalog data |
| `rentals.txt` | Rental records data |
| `users.txt` | User account data |
